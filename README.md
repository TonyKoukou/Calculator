# Simple Calculator


## Περιγραφή
Python πρόγραμμα με Tkinter για απλές μαθηματικές πράξεις.

## Οδηγίες Χρήσης
1. Βεβαιώσου ότι έχεις εγκατεστημένη Python 3.
2. Άνοιξε το πρόγραμμα:cmd
3. Γράψε cd και την τοποθεσία του αρχείου π.χ cd C:\Users\Tony\Calculator
4. Γράψε την εντολή: python calculator.py
5. Εισάγετε αριθμούς στο πεδίο και πατήστε "Calculate".
6. Δες το αποτέλεσμα στο παράθυρο.
